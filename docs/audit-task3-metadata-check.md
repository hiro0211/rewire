### ⚠️ 年齢制限に関する懸念

現在 **12+/13+** だが、アプリの内容は以下の理由で **17+** が妥当な可能性がある:

1. **アプリ名とサブタイトルに「ポルノ」が明示的に含まれる**
2. **説明文にポルノ視聴・ポルノ依存・ポルノサイトの記述が多数ある**
3. **Apple Guideline 2.3.6**: 年齢制限の質問票で「頻繁な/強い性的なコンテンツまたはヌード」に該当しないが、「まれな/軽度の成人向けテーマ」には該当する可能性
4. **ただし**: アプリ自体はポルノコンテンツを含まず、**回復/ヘルスケア**が目的。類似アプリ（Brainbuddy, Fortify等）は12+〜17+で分かれている

**判断**: 保守的に **17+** に変更することを強く推奨。レビューで年齢制限の不適切さを指摘されるリスクがある。

### ❌ 広告データ — トラッキング設定の問題

**問題**: 「広告データ」が「トラッキング目的に使用」として設定されている。

**影響**:
- ATT（App Tracking Transparency）は使用しない → NSUserTrackingUsageDescription を削除済み
- トラッキング目的でデータを収集するなら、ATTダイアログが**必須** (Guideline 5.1.2)
- ATTなしで「トラッキング」を申告すると矛盾が生じ、**リジェクトリスク**

**対応**:
- 「広告データ」のトラッキング設定を**オフ**にする
- または「広告データ」自体を削除する（アプリ内に広告は存在しない）

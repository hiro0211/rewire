## 【衝撃】大学がコロナで封鎖された3年間で、何が起こったか

[画像: 夕焼けの大学キャンパスが人影もなく静寂に包まれている。図書館の窓から薄く光が漏れている]

どうも、有村です！

突然ですが、想像してみてください。

大学2年の春、キャンパスは突然、閉ざされた。友達に会えない。講義はZoom。外出も気がひける。世界全体が引きこもりモードになり、僕の部屋も牢獄に変わった。

その先に待っていたのは、**集中力の完全な崩壊**でした。

---

**▼ 小学5年生からの「付き合い」**

実は、僕がポルノと出会ったのは、もっと前です。小学5年生。父親のブラウザ履歴から偶然見つけた。その時は「へー、大人ってこんなの見るんだ」くらいの感覚だった。

でも中学に入ると、思春期のホルモンと好奇心が相まって、夜中に何度も。高校では受験勉強の「気分転換」として。大学では、一応は「大人だからいいでしょ」という言い訳。

いつの間にか、1日1回が当たり前になってた。

```
この頃の僕は気づいていなかった。
毎日の習慣が、脳を蝕んでいたことに。
```

---

**▼ コロナが「スイッチ」を入れた**

2020年4月。大学2年。緊急事態宣言。

「家にいろ」という国の命令で、僕は自分の部屋に監禁された。もちろん、ネット環境だけは完璧だ。むしろ時間は無限にある。

朝起きて、スマホを手に取った瞬間——もう止まりませんでした。

- 朝8時：起床して30分で1回目
- 昼12時：お昼ご飯の後に1回
- 夕方17時：「気分転換」という名目で1回
- 夜23時：寝る前に1回

1日平均3回。多いときは5回。

Zoomの講義中でさえ、後ろの方に隠れてスマホを触ってた。恥ずかしい話ですが、それが現実です。

同時に、何が起こったか。

---

**▼ 脳が、壊れていった**

集中力が消えた。本当に、消えた。

レポートを書こうと思って、パソコンを開く。5分後、スマホを手に取ってる。10分後、ブラウザを開いてる。気づいたら1時間が無くなってる。

試験勉強？まともにできません。教科書を読んでも、3行で内容が入らない。前頭前皮質（意思決定と集中力の中枢）が、ドーパミン報酬系の暴走で完全に制御を失ってたんです。

大学の成績は、一気に下がった。GPA 3.8から、1学期で2.3まで落ちました。

```
「このままじゃ、ダメな大人になる」

心のどこかで、そう思ってた。
でも、やめられなかった。
```

やめようとすると、イライラ。禁断症状です。ドーパミンが枯渇してるから、他の快楽（食事、映画、ゲーム）もつまらなく感じる。結局、スマホに戻る。

その悪循環が、3年間続きました。2020年4月から2023年3月まで。

コロナはずっと続いたわけじゃないけど、一度壊れた習慣パターンは、オンライン授業が常態化した大学後半もずっと続いた。

---

**▼ 「実は、俺もだ」という友人の一言**

転機は、ある友人の何気ない告白でした。

Discordで話していたとき、彼が言ったんです。

```
「有村、俺、今年からポルノやめてるんだけど…」
```

僕は、自分の状況を打ち明けました。彼は笑わなかった。むしろ、こう言ってくれた。

```
「俺も同じ。でもな、やめたら人生変わるよ。
集中力が戻ってきた。彼女もできた。
お前も試してみよ」
```

その友人は、有名な研究論文を見せてくれました。

**ドーパミン報酬系の可塑性に関する神経科学の研究では、ポルノ視聴による過度な刺激は、自然なドーパミン応答を鈍くさせる。つまり、日常の喜びに鈍感になる。ただし、やめると3ヶ月〜6ヶ月で脳が「リセット」される、という報告があります。**

その時、初めて僕は思った。

```
「これ、自分の弱さじゃなくて、
脳科学の問題じゃん」
```

---

**▼ 120日間の静かな戦い**

2023年3月、友人の言葉に背中を押されて、本気でやめることに決めました。

ただし、「絶対にやめる！」みたいな気合いは入れませんでした。むしろ逆です。

```
「完璧にやめなくていい。
続いている実感が大切」
```

その言葉を信じて、毎日を数えることにしました。

1日目。2日目。10日目。30日。

衝動は来ます。特に寝る前、ストレスを感じたとき。でも、その時に僕がやったのは：

- **呼吸を整える**（4秒吸って、6秒かけてゆっくり吐く。副交感神経を優位にする）
- **スマホを部屋の別の場所に置く**（環境設計）
- **友人に「今日も続いてる」とメッセージを送る**（責任感）

60日を超えたあたりから、脳が変わり始めました。

本が読める。集中力が戻ってきた。1時間、連続して何かに集中できるようになった。

90日を超えたときは、驚きました。朝起きた時の「やる気」が違う。以前の自分は、朝から虚無感でいっぱいだったのに、今は「今日、何をしようかな」という前向きな感覚がある。

120日。4ヶ月。

彼女もできました。いや、マジで。これ、ポルノをやめた副産物です。集中力が戻ると、対人スキルも磨かれるんですよね。目が，相手を見つめられるようになる。会話に集中できるようになる。

---

**▼ なぜ、これを書くのか**

「有村、なんでこんなプライベートなこと、noteに書くの？恥ずかしくないの？」

そう聞かれます。

でも、僕は思うんです。**自分と同じ悩みを持つ人は、絶対にいる。** 20代男性の50%以上が、何らかの形でポルノ依存の悩みを抱えているという調査結果もあります。

でも、誰も話さない。恥ずかしいから。

だから、僕が話す。そして、「やめたら人生変わるよ」って、伝える。

昔の僕は、「やめられない自分」を責めていました。意思が弱いんだと。でも違った。**これは、意思の問題じゃなくて、脳科学の問題で、正しい知識と環境設計があれば、誰でも変えられます。**

---

僕たちは、変わることができます。

皆さんは、自分の人生で「やめたいのにやめられない」という習慣を持っていますか？

## Rewireアプリについて

この記事で書いたような「毎日の衝動との戦い」と「続いている実感を作る」ことをサポートするために、僕が作ったアプリがあります。

**Rewire — ポルノをやめたいあなたのためのアプリ**

ストリーク記録で「続いている実感」が見える。毎日の進捗が目に見えるから、モチベーションが続く。

衝動が来たときは、SOS呼吸法（4秒吸って、6秒かけてゆっくり吐く）で、ピークをやり過ごせる。

デイリーチェックインで、その日の誘惑レベルや生活の質を記録。データを見返すと、「ポルノをやめた日の方が、仕事もプライベートも充実してる」という気づきが得られます。

僕自身が「こんなアプリがあったら良かったのに」と思いながら、大学時代を過ごしていたから、その経験を全部詰め込みました。

無料版でも、ストリーク記録とSOS呼吸法は使えます。

## 1. プロダクトビジョン

### 1.1 一言で

> **ポルノに到達する前に止める。脳を再配線し、人生を前に進めるアプリ。**

### 1.2 コンセプトステートメント

Rewire は、ポルノをやめたい日本人男性が **SNS → ドーパミンエスカレーション → ポルノ視聴** という行動チェーンを科学的に断ち切り、節約した精神リソースをキャリア・副業・自己成長に再投資できる状態を作るアプリである。

ポルノをやめること自体は目的ではない。その先にある **「自分をコントロールできている実感（自己効力感）」と「自分との約束を守れる人間になること」** がゴールである。

### 1.3 プロダクト原則（守るべき判断基準）

| # | 原則 | 意味 |
|---|------|------|
| 1 | **上流で止める** | ポルノサイトをブロックするだけでは遅い。SNSを衝動的に開く瞬間に介入する |
| 2 | **本能ではなく理性にスイッチさせる** | 意志力に頼らない。行動科学のフリクション設計で衝動の連鎖を構造的に断つ |
| 3 | **ポルノ断ちは手段、自己成長が目的** | 「やめる」ではなく「前に進む」。キャリア・副業・集中力の文脈で設計する |
| 4 | **弱者扱いしない** | 依存症、治療、回復といった言葉は使わない。自分を律して前に進む人向け |
| 5 | **マスターベーションは否定しない** | ポルノ視聴に特化する。自然な性欲まで抑圧する設計にしない |

## 4. 課題の構造分析 — ドーパミンチェーン

### 4.1 ポルノに到達する行動チェーン

中村 拓也がポルノにたどり着くパターンは「衝動的にポルノサイトを開く」ではない。以下のチェーン（連鎖）を経由して段階的にエスカレーションしている。

```
[Stage 0] ストレス・退屈・疲労
    │
    │  ドーパミンを求めてスマホに手を伸ばす
    ▼
[Stage 1] SNSを衝動的に開く（Instagram, TikTok, YouTube Shorts）
    │
    │  アルゴリズムが性的コンテンツをフィードに表示
    ▼
[Stage 2] 性的コンテンツに遭遇（リール、ショート動画）
    │
    │  投稿主のプロフィールを閲覧
    ▼
[Stage 3] プロフィール閲覧 → 他の投稿を巡回
    │
    │  刺激が物足りなくなり、より強い刺激を求める
    ▼
[Stage 4] Safari を開き、ポルノスターの作品を検索
    │
    ▼
[Stage 5] ポルノサイトで視聴
    │
    ▼
[Stage 6] 視聴後の後悔・集中力壊滅・自己効力感低下
```

### 4.2 既存ソリューションの限界

| ソリューション | 介入ポイント | 限界 |
|--------------|-------------|------|
| Safariアダルトブロック（iOS標準） | Stage 4〜5 | 自分でパスコード入力して解除可能。性的衝動下では解除する |
| Brainbuddy | Stage 5（衝動時の呼吸）＋ 記録 | ポルノに到達してから介入。SNSからの流入を止められない |
| one sec | Stage 1（SNS起動時のフリクション） | ポルノ特化ではない。文脈理解なし。目標・理性スイッチングがない |

### 4.3 Rewire の介入設計 — 3層防御

```
[Layer 1] SNSフリクション          ← Stage 1 で介入 ★ 最重要
    ↓ 突破した場合
[Layer 2] Safariブロッカー          ← Stage 4 で介入
    ↓ 解除しようとした場合
[Layer 3] 理性スイッチング＋呼吸    ← Stage 4〜5 で介入
    ↓ それでも見てしまった場合
[Recovery] リカバリー導線           ← Stage 6 で回収
```

**核心的な差別化：** 既存アプリはすべて Stage 4 以降（ポルノに到達した後 or 到達しようとしたとき）に介入する。Rewire は **Stage 1（SNSを開く瞬間）** に介入し、ポルノに到達するチェーン自体を発生させない。


## 7. 科学的根拠に基づく介入設計

### 7.1 Layer 1: SNSフリクション介入の科学的根拠

**根拠①：one sec のPNAS掲載論文**

マックスプランク研究所・ハイデルベルク大学との共同研究（2023年、PNAS掲載）。280名を対象に6週間の実験を実施。SNSアプリ起動前にフリクション（待機時間）＋ 熟考メッセージ ＋ 離脱オプションの3要素を組み合わせた結果、アプリの実際の起動回数が57%減少した。最も効果が高かったのは「離脱オプションの表示」であり、次に「フリクション（時間遅延）」が効果的だった。

Rewire への応用：SNSを開こうとした瞬間に呼吸インターバル（フリクション）→ 目標リマインド（熟考メッセージ）→ 「開かずに戻る」ボタン（離脱オプション）の3要素を実装する。

**根拠②：ドーパミンの刺激制御（Stimulus Control）**

CBTベースの刺激制御の原則。望ましくない刺激をアクセス困難にし、代替行動を用意することで衝動行動を減らす。Rewire ではSNSの起動にフリクションを加えることで、ドーパミンチェーンの起点（Stage 1）を弱体化させる。

### 7.2 Layer 3: 衝動介入の科学的根拠

**根拠③：Urge Surfing（衝動サーフィン）**

依存症研究の先駆者 Alan Marlatt博士が提唱した技法。衝動を「波」として観察し、戦わずにやり過ごす。研究によると、衝動は対処しなくても20〜30分でピークを越えて自然に減衰する。衝動に対して好奇心を持って観察する態度を取ると、感情的反応性が低下し、衝動への耐性が高まる。

Rewire への応用：深呼吸ガイドで衝動のピーク（最初の数分間）をやり過ごす時間を稼ぐ。呼吸に没入させることで、衝動と行動の間に「間（ま）」を作り、ハビットループ（トリガー → 行動 → 報酬）を中断する。

**根拠④：Implementation Intentions（実行意図）**

Gollwitzer（1999）が提唱した「if-then プラン」。「もし〜の状況になったら、〜する」を事前に決めておくことで、状況と代替行動の連想が自動化され、習慣的反応を上書きできる。

Rewire への応用：オンボーディングで「衝動が来たらSOSボタンを押す」を設定させる。これにより「衝動 → ポルノ」の自動反応を「衝動 → SOSボタン → 呼吸」に書き換える。

### 7.3 回復の科学的タイムライン

| 期間 | 変化 |
|------|------|
| 最初の1〜2週間 | 最も衝動が強い。睡眠・気分が不安定になりやすい |
| 3〜6週間 | トリガーの力が弱まり始める。新しいルーティンが定着し始める |
| 2〜3ヶ月 | 注意力・エネルギー・気分がより安定する |
| 6〜12ヶ月 | 突発的な衝動が大幅に減少。アイデンティティの変化（「ポルノを見ない人」としての自己認識） |

これは中村 拓也の「13日の壁」を説明する。最初の2週間が最も困難であり、そこを突破できれば3〜6週間目から急速に楽になる。Rewire は最初の2週間を生き延びるための強力な支援ツールとして機能する。

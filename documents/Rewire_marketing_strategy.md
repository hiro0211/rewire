## 1. 戦略の原点 — hiroの原体験

### 1.1 hiroがポルノ断ちに至った経路

```
恋愛屋ジュン、メンタルコーチ ジョージ（YouTube）を視聴
    ↓
ポルノが脳に与える悪影響を「教育コンテンツ」で知る
    ↓
「自分もやめなければ」と課題を自覚する
    ↓
Brainbuddy等の競合アプリを検索・試用する
    ↓
「日本語で、SNS起因の行動チェーンに対応するアプリがない」と気づく
    ↓
Rewireを自分で作る
```

### 1.2 この原体験が教えてくれること

hiroがアプリを探したのは、バズ動画を見たからではない。真面目な教育コンテンツで課題を自覚し、自分から解決策を探しに行った。この行動パターンこそが「課金する顧客」の行動パターンである。

つまり、Rewireのマーケティングは「過去のhiroと同じ人」を見つけ出し、課題を自覚させ、Rewireという解決策に導くことが本質である。

### 1.3 マーケティングの基本方程式

```
教育コンテンツで課題を自覚させる × ASOでRewireを発見可能にする = 課金顧客
```

バズ動画で100万人に届けて15人が課金するより、真面目な教育で5万人に届けて75人が課金する方が価値がある。

## 2. マーケティング目標

### 2.1 ビジネスゴール

| 指標 | 目標 | 根拠 |
|------|------|------|
| サブスク加入者（Pro） | 6ヶ月で月間500人 | 質の高い顧客に絞る |
| CVR（無料→有料） | 15%以上 | 真面目戦略は業界平均の3倍のCVRが見込める |
| LTV | 年間5,760円以上（月額480円 × 12ヶ月） | 課題を自覚した顧客は継続率が高い |
| 月次解約率 | 5%以下 | 教育経由の顧客は解約しにくい |

### 2.2 マーケティングゴール

| 指標 | 目標 | 優先度 |
|------|------|--------|
| ASO「ポルノ やめる」で1位 | 3ヶ月以内 | ★★★ 最優先 |
| ASO「ドーパミン リセット」で1位 | 6ヶ月以内 | ★★★ |
| TikTok教育コンテンツ蓄積 | 100本 | ★★★ |
| UGC体験談の蓄積 | 月間30件 | ★★☆ |
| note記事のSEO流入 | 月間1,000PV | ★★☆ |

**注意：** フォロワー数は目標にしない。フォロワー数は虚栄指標（Vanity Metric）であり、課金に直結しない。

## 3. リサーチから得た知見

### 3.1 Quittr — 4ヶ月で月間$250,000のMRR

| 手法 | 詳細 |
|------|------|
| ニッチインフルエンサー | キリスト教系、フィットネス系、自己啓発系の3ニッチ |
| 初期投資$50 | 最初のインフルエンサーへの報酬$50の通話 → 日商$1,000 |
| メガバイラル | 200万フォロワーのインフルエンサー → 990万再生 → 1日$40,000 |
| 動画ルール | アプリUI画面を必ず含める。広告転用OK |
| ハードペイウォール | 年額$45。高い継続率 |
| 広告スケール | Meta/Instagram広告に月$40,000 → ROAS 3-4倍 |

**Rewireへの教訓：** Quittrの成功は英語圏のスケールがあってこそ。日本市場ではインフルエンサーの数が限られるため、hiro自身が教育者ポジションを取る方が確実。ただしPhase 2以降でニッチインフルエンサー活用は検討する。

### 3.2 PuffCount — TikTokマーケティングで月商640万円

| 手法 | 詳細 |
|------|------|
| エンタメ重視 | セールス動画ではなくエンタメとして消費される動画 |
| UGC風 | プロっぽくしない。カジュアルに制作 |
| CVR 25% | ハードペイウォール + 無料トライアル |
| 価格ABテスト | 640円〜1,920円でテストしLTV最適化 |

**Rewireへの教訓：** 「マーケティングが成功の95%」は真実。ただしPuffCountのエンタメ路線ではなく、教育路線でCVRを最大化する。

### 3.3 恋愛屋ジュン・メンタルコーチ ジョージ — hiroの原体験

| 特徴 | 詳細 |
|------|------|
| コンテンツスタイル | 真面目な教育。科学的根拠に基づいた解説 |
| ターゲット | 20-30代男性。自己成長意欲が高い |
| 収益モデル | 教育コンテンツ → 信頼構築 → 商品・サービスへの導線 |
| フォロワーの質 | 課題を自覚した人が集まる。「面白い」ではなく「役に立つ」で選ばれる |

**Rewireへの教訓：** hiro自身がこのポジションを取る。ジュンやジョージのように真面目な教育者として発信し、その延長線上にRewireアプリがある構造を作る。

### 3.4 令和ノート（岡本龍生氏）

| 手法 | 詳細 |
|------|------|
| TikTok寸劇バイラル | 「令和ノートを使うことを禁止します」寸劇がバイラル |
| ステマ疑惑 | UGC風だがステマの指摘あり |

**Rewireへの教訓：** 寸劇バイラルはリーチは取れるがCVRが低い。Rewireのテーマ（ポルノ断ち）は寸劇向きではなく、真面目路線の方が顧客の質が高い。ステマは絶対にしない。

## 4. コアファネル設計

### 4.1 過去のhiroを再現するファネル

```
[Step 1] 教育コンテンツに出会う
          TikTok / note / X で「ポルノが脳に与える影響」を知る
          → hiro自身のコンテンツ or インフルエンサーのコンテンツ
              ↓
[Step 2] 課題を自覚する
          「自分もドーパミンチェーンにハマっている」と気づく
          → コンテンツの中で自己診断できる問いかけを入れる
              ↓
[Step 3] 解決策を探す
          App Storeで「ポルノ やめる」「ドーパミン リセット」と検索
          → ASOで1位を取り、Rewireが最初に見つかる状態にする
              ↓
[Step 4] Rewireをダウンロードする
          無料で基本機能を体験。チェックイン、呼吸介入
              ↓
[Step 5] 価値を実感する
          7日ストリーク達成。「今までと違う」と感じる
              ↓
[Step 6] Pro課金する
          3層防御を有効化。本格的にポルノ断ちに取り組む
              ↓
[Step 7] ストリークを共有する（UGC）
          達成をシェア → 同じ悩みを持つ人がコンテンツを見つける → Step 1に戻る
```

### 4.2 ファネルの肝

| ポイント | 解説 |
|----------|------|
| Step 1-2がマーケティングの全て | 課題を自覚した人は自分から動く。ASOさえ整っていればDLに至る |
| Step 3のASO独占が参入障壁 | 「ポルノ やめる」で1位 = 後発がどれだけ広告を打っても自然検索で勝てない |
| Step 7のUGCが複利で効く | ユーザーが新しいユーザーを連れてくる。コスト0の顧客獲得 |

## 5. TikTokアカウント設計

### 5.1 アカウントの方向性：「教育者」ポジション

バズを狙うエンタメアカウントではなく、恋愛屋ジュンやメンタルコーチ ジョージと同じ「真面目な教育者」ポジションを取る。

| 項目 | 設計 |
|------|------|
| アカウント名 | @rewire_jp |
| 表示名 | hiro｜ポルノが脳を壊す仕組みを解説 |
| プロフィール | 🧠 ポルノの脳への影響を科学的に解説 ／ 📊 自分の体験から学んだこと ／ アプリ開発者 → リンク |
| トーン | 真面目・誠実・科学的。弱者扱いしない。「一緒に前に進もう」 |
| 顔出し | 任意。顔なし（声+テキスト+図解+データ）でも成立する |
| 目的 | バズではなく「この人の話は信頼できる」と思わせること |

### 5.2 コンテンツの4カテゴリ

**A. 科学教育（50%）— メインコンテンツ**

課題を自覚させるコンテンツ。過去のhiroが「これ見て目が覚めた」と思えるもの。

| テーマ例 | フック |
|----------|--------|
| ドーパミンの仕組み | 「ポルノを見た後やる気が出ない理由、脳科学で説明する」 |
| ヘロインとの類似性 | 「ポルノとヘロイン、脳のスキャン画像がほぼ同じだった研究」 |
| 集中力の破壊 | 「ポルノを見た日、なぜ本が読めなくなるのか」 |
| SNS→ポルノのチェーン | 「Instagram→TikTok→ポルノ。このチェーン、心当たりある人」 |
| 前頭前皮質の萎縮 | 「脳の前頭前皮質が縮む。意志力が物理的に弱くなる仕組み」 |
| ドーパミン基準値 | 「なぜ同じポルノじゃ満足できなくなるのか。脳の耐性の仕組み」 |

制作ルール：
- 最初の1秒で「知らなかった事実」を提示する
- 科学論文や研究データを根拠にする
- 30-90秒。情報密度を高くする
- 「セールス」は一切しない。純粋な教育コンテンツ
- 画面構成：テキスト + 図解 + データ。顔出し不要

**B. 体験談（25%）— 信頼構築コンテンツ**

hiro自身の体験、またはユーザーの許可を得た体験談。

| テーマ例 | フック |
|----------|--------|
| hiro自身の13日の壁 | 「13日が限界だった。14日目に何が起きたか話す」 |
| 回復の実感 | 「30日やめたら、本が1冊読めるようになった」 |
| 失敗と学び | 「昨日、衝動に負けた。何がトリガーだったか振り返る」 |
| ビフォーアフター | 「3ヶ月前の自分と今の自分を比較する」 |

制作ルール：
- 嘘をつかない。盛らない。リアルな体験だけを語る
- 失敗も包み隠さず出す。完璧な人の話は誰も信じない
- 「私の場合」を前提にし、押し付けない

**C. How-to（15%）— 解決策コンテンツ**

具体的にどうすればポルノをやめられるかの方法論。Rewireの導線になる。

| テーマ例 | フック |
|----------|--------|
| 衝動への対処法 | 「衝動が来た時の3ステップ。科学的に効果がある方法」 |
| SNSとの距離の取り方 | 「SNSを開く前に10秒だけ○○をする。それだけで57%減」 |
| 夜の時間の使い方 | 「22時以降のスマホの使い方を変えるだけで人生が変わる」 |
| チェックインの効果 | 「毎日1分の振り返りが、なぜ衝動を減らすのか」 |

制作ルール：
- 一般的なアドバイスとして語る。「Rewireを使え」とは言わない
- ただしコメントやプロフィールで「具体的にはアプリを作りました」と自然に誘導
- 「じゃあどうすればいいの？」と聞かれた時の回答としてRewireがある状態を作る

**D. データ・ファクト（10%）— 権威構築コンテンツ**

| テーマ例 | フック |
|----------|--------|
| ユーザーデータ | 「1,000人分のデータ。最も衝動が強い曜日と時間帯」 |
| 統計 | 「日本人男性の○%がポルノの影響を自覚している」 |
| 開発の裏側 | 「なぜSNSを開く瞬間に介入するアプリを作ったのか」 |

### 5.3 投稿スケジュール

| 頻度 | 内容 |
|------|------|
| 平日（月-金） | 1日1本。科学教育 or 体験談 |
| 土日 | 任意。How-to or データ系 |
| 目標 | 月20-25本。量より質を優先 |

### 5.4 TikTokからの導線設計

```
TikTok動画（教育コンテンツ）
    ↓ コメント欄で質問が来る
プロフィールリンク → Rewireアプリ or note記事
    ↓
App Store → DL → 体験 → 課金
```

**動画内でアプリを直接宣伝しない。** 教育コンテンツの信頼が貯まれば、ユーザーは自分からプロフィールを見にくる。セールス感を出した瞬間に教育者としての信頼が落ちる。

## 6. チャネル別戦略

### 6.1 チャネル優先順位

```
[Tier 1] TikTok        ← 教育コンテンツの主戦場。課題認識を生む
[Tier 2] note           ← 長文教育 + SEO。App Store検索の前段階
[Tier 3] X              ← 個人開発者ブランド。データ公開。顧客との対話
[Tier 4] Instagram      ← UGCの集約。コミュニティの可視化
```

### 6.2 note — SEO + 教育の深掘り（Tier 2に格上げ）

noteは「教育→課題自覚→検索」ファネルにおいて極めて重要。TikTokでは伝えきれない深い内容をnoteで書き、Google検索経由で課題を自覚した人を拾う。

| テーマ | 内容 | 狙うキーワード |
|--------|------|--------------|
| 科学解説 | ポルノが脳に与える影響の網羅的まとめ | 「ポルノ 脳 影響」 |
| 体験談 | hiro自身のリアルな体験（匿名でも可） | 「ポルノ やめた」「ポルノ断ち 体験談」 |
| 方法論 | 具体的にどうやめるかのステップ | 「ポルノ やめ方」「ドーパミン リセット 方法」 |
| アプリ解説 | Rewireの設計思想と科学的根拠 | 「ポルノ やめる アプリ」 |

頻度：週1本。質を優先。1記事3,000-5,000字。

### 6.3 X — 個人開発者ブランド

| 内容 | 頻度 |
|------|------|
| 開発データ公開（DL数、CVR、MRR） | 週2回 |
| ユーザーの声（許可済み） | 週1回 |
| 科学的ファクトの短文投稿 | 週2回 |
| 個人開発の学びや気づき | 適宜 |

Xの役割は「hiro＝Rewire開発者」のブランドを構築すること。個人開発者コミュニティとの接点にもなる。

### 6.4 Instagram — UGCの可視化

| 内容 | 頻度 |
|------|------|
| ユーザーのストリーク共有をリポスト | 随時 |
| カルーセル（科学解説の図解版） | 週2回 |
| リール（TikTokコンテンツの転載） | 週3回 |

Instagramは自分から発信するよりも、UGCの集約地として活用する。

## 7. UGC戦略

### 7.1 UGCの目的

バズではなく以下の3つを目的とする：

| 目的 | 具体的な価値 |
|------|------------|
| ソーシャルプルーフ | 「他の人もやっている」という証拠。App Storeレビューの代替 |
| 参入障壁 | 数千件のUGCは後発がコピーできない |
| 顧客獲得コスト0 | ユーザーが勝手に宣伝してくれる |

### 7.2 UGCを自然に生む仕組み

| 仕組み | 詳細 |
|--------|------|
| ストリーク達成画像の自動生成 | 7日、14日、30日、90日達成時に美しいシェアカードを生成 |
| デザインの質 | ダークテーマ + ゴールドアクセント。「これ共有したい」と思えるデザイン |
| ハッシュタグ統一 | #Rewire挑戦 をメインタグとして定着させる |
| 匿名可 | 顔出し・名前出し不要。ポルノ断ちはデリケートなテーマ |
| テンプレート提供 | 「○日目。今の状態は○○。変化は○○」のテンプレ |
| リポスト文化 | 公式アカウントでリポスト。ユーザーの承認欲求を満たす |

### 7.3 UGCの質の管理

| ルール | 理由 |
|--------|------|
| ステマは絶対にしない | 一度バレたら信頼は二度と戻らない |
| 対価を払ったUGCにはPR表記必須 | 法的義務 + 倫理 |
| ネガティブUGCも許容 | 「失敗した」もリアルなUGC。コミュニティの信頼を高める |
| 個人情報は含めない | ストリーク数と簡単なコメントだけ |

## 8. インフルエンサー戦略

### 8.1 基本方針：Phase 2以降で段階的に

Phase 1はhiro自身のコンテンツで検証する。Phase 2でインフルエンサーに展開する。

### 8.2 ターゲットニッチ（日本市場）

| ニッチ | 理由 | 例 |
|--------|------|-----|
| 自己啓発・自己管理系 | ポルノ断ちを「自己コントロール」の文脈で語れる | 恋愛屋ジュン系の教育者 |
| 筋トレ・ボディメイク系 | テストステロン、集中力との親和性 | 禁欲×筋トレ系のアカウント |
| ビジネス・副業系 | 「時間とエネルギーの再投資」の文脈 | 20代副業チャレンジ系 |
| メンタルヘルス系 | ドーパミンリセット、デジタルデトックス | メンタルコーチ ジョージ系 |

### 8.3 選定基準

| 基準 | 詳細 |
|------|------|
| 規模 | マイクロ（1,000-10,000）が最もCVR高い |
| トーン | 真面目系。煽り系やエンタメ系は除外 |
| フォロワー属性 | 20-40代男性中心 |
| 信頼性 | ステマ歴なし。誠実な発信者 |

### 8.4 報酬設計

| フェーズ | 報酬 |
|----------|------|
| テスト | Pro版無料提供 + 成果報酬（インストール数） |
| 検証済み | 固定報酬1-3万円 + 成果報酬 |
| スケール | 月間契約 + 広告転用権 |

### 8.5 動画ルール

| ルール | 詳細 |
|--------|------|
| RewireのUI画面を必ず含める | アプリの存在と使い方を視覚的に伝える |
| PR表記必須 | #PR #広告 を必ず入れる |
| インフルエンサー自身の言葉で語る | 台本を押し付けない |
| 広告転用権 | 動画をMeta/TikTok広告に使用できる契約 |

## 9. ASO戦略と参入障壁

### 9.1 ASO（App Store Optimization）— 最重要施策

hiro自身のファネルが「課題自覚→App Store検索→DL」だったように、ASOは最も重要な参入障壁である。

### 9.2 ターゲットキーワード

| キーワード | 優先度 | 現状の競合 |
|-----------|--------|-----------|
| ポルノ やめる | ★★★ | 競合少ない。独占可能 |
| ポルノ断ち アプリ | ★★★ | 競合ほぼなし |

## 10. CVR・LTV最大化の設計

### 10.1 CVR最大化のポイント

「教育経由の顧客」はCVRが高い理由：

| 理由 | 解説 |
|------|------|
| 課題を自覚済み | 「自分はポルノを断つべき」と決めている状態でDLする |
| 解決策を能動的に探した | 押し売りではなく自分から来た顧客 |
| 科学的根拠に共感済み | ドーパミンの仕組みを理解した上で使い始める |
| 開発者を信頼済み | TikTok/noteで教育コンテンツを見て信頼した状態 |

### 10.2 価格戦略

| プラン | 価格 | 目的 |
|--------|------|------|
| 月額 | 480円 | 低ハードル |
| 年額 | 3,800円（月317円） | LTV最大化。年額への誘導が最優先 |
| 生涯 | 9,800円 | キャッシュフロー。限定オファー時のみ |

ABテスト項目：
- 月額480円 vs 580円 vs 780円
- 年額プランの割引率
- 無料トライアル期間（3日 vs 7日）
- ペイウォールタイミング（7日達成時 vs 14日達成時）

### 10.3 解約防止

| 施策 | 詳細 |
|------|------|
| ストリーク表示 | 「○日連続を手放しますか？」 |
| 一時停止オプション | 解約ではなく休止を提案 |
| データ保持 | Free版でもデータは保持。いつでも復帰可能 |
| 解約はスムーズに | 引き留めは最小限。しつこくしない |

## 13. 禁止事項

CLAUDE.mdの原則とプロダクトコンセプトシートに基づく。

| 禁止事項 | 理由 |
|----------|------|
| ステルスマーケティング | 一度バレたら信頼は二度と戻らない。PR表記必須 |
| バズ目的のエンタメコンテンツ | 課金しない視聴者を集めても意味がない |
| ダークパターン課金 | 価値を先に届け、納得の上で課金してもらう |
| 失敗直後の課金誘導 | ポルノ視聴後のユーザーにセールスしない |
| 虚偽の緊急性 | 嘘のカウントダウン、架空の限定は使わない |
| 解約妨害 | スムーズに解約できること |
| 恐怖マーケティング | 「人生終わる」式の恐怖煽りを主軸にしない |
| 依存症・患者扱い | 弱者扱いしない |
| 性的コンテンツの使用 | マーケティング素材に性的画像は使わない |
| 個人情報の無断公開 | 体験談は必ず許可を得る |
| 他アプリの誹謗中傷 | 競合を貶さない |
| フォロワー数の追求 | 虚栄指標を目的にしない |
